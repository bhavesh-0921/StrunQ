# StrunQ 🎸

A free, browser-based virtual guitar platform that lets you play like a real guitarist — no instrument, no downloads, no account needed.

Set up your capo, build a chord progression, define your own strumming pattern (D, U, d, u notation), set your BPM, and hit play. StrunQ handles the rest.

---

## Why StrunQ?

Most people who want to learn guitar never start. The reasons are always the same — the instrument is too expensive, the learning curve is too steep, and existing online tools are either too limited or too hard to use.

StrunQ was built by someone who felt that exact frustration. This is the tool we wished existed.

---

## Features

- 🎸 Realistic guitar sound via browser audio synthesis
- 🎵 Custom strumming pattern builder — D, U, d, u notation like real guitarists use
- 🎼 Full chord progression builder — G, C, D, Am, Em, F, Dm, E, A, Bm
- 🎚️ Capo position selector (0–7) with real-time chord transposition
- ⏱️ BPM control with tempo presets
- 📱 Mobile friendly — works on any device, any browser
- 🆓 Completely free — no login, no download, no hardware

## Tech Stack

- **React** (Vite)
- **Tone.js** — browser audio engine
- **Tailwind CSS** — styling
- **Vercel** — hosting

## Getting Started Locally
```bash
git clone https://github.com/bhavesh-0921/StrunQ.git
cd StrunQ
npm install
npm run dev
```
